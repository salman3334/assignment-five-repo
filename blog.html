Question-1: What are the different ways to select an element in the DOM?

The different ways to select an element in the DOM are: By ID: document.getElementById("id") By Class Name: document.getElementsByClassName("class") By Tag Name: document.getElementsByTagName("tag") By CSS Selector: document.querySelector("selector") By CSS Selector (Multiple): document.querySelectorAll("selector")

Question-2: Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML – Returns or sets the HTML content inside an element, parsing HTML tags. innerText – Returns or sets only the visible text, ignoring hidden elements and extra spaces. textContent – Returns or sets all text, including hidden elements, without parsing HTML.

Question-3: Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where a parent element handles events for its child elements, even if they are added dynamically. It uses event bubbling to improve performance and reduce the number of event listeners.

Question-5: How do you create, add, and remove elements using JavaScript?

Create an element: document.createElement("tag") Add an element: parent.appendChild(child) or parent.append(child) Insert before another element: parent.insertBefore(newElement, referenceElement) Remove an element: element.remove() or parent.removeChild(child)
